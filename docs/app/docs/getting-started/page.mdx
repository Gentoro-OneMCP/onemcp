# Getting Started

<Steps>
### Install CLI

First, add the OneMCP bucket:

<Tabs items={['Homebrew', 'scoop']} defaultIndex="0" >
  <Tabs.Tab>
      Add OneMCP formula:

      ```bash
      brew tap gentoro-onemcp/onemcp
      ```

      Then install the CLI:

      ```bash
      brew install onemcp
      ```
  </Tabs.Tab>
  <Tabs.Tab>
    Add OneMCP bucket:

    ```bash
    scoop bucket add onemcp https://github.com/gentoro-onemcp/onemcp-scoop
    ```

    Then install the CLI:

    ```bash
    scoop install onemcp
    ```
  </Tabs.Tab>
</Tabs>

### Create OneMCP Project

```bash
# Create a new project directory
mkdir ~/my-onemcp-project && cd ~/my-onemcp-project

# Initialize the project within the current directory
# Alternatively, you can run: onemcp init --project-dir ~/my-onemcp-project/
onemcp init
```
